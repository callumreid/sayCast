# Homebrew Tap for sayCast

This tap hosts the Homebrew formula for installing sayCast.

## Installation

```bash
brew tap yourusername/homebrew-saycast
brew install saycast
```

## Upgrade

```bash
brew update
brew upgrade saycast
```

## Uninstall

```bash
brew uninstall saycast
brew untap yourusername/homebrew-saycast
```
